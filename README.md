# Product Catalog

## Описание решения

В рамках данного проекта реализован каталог товаров с возможностью поиска, фильтрации, сортировки, пагинации и добавления товаров в корзину. Архитектура и стек технологий были выбраны исходя из требований ТЗ, с задумкой на масштабируемость, читаемость кода и соответствие современным практикам frontend-разработки.
---

## Выбор технологий и обоснование

**Vite**
Vite выбран из‑за высокой скорости разработки, простой конфигурации по сравнению с Webpack.

**Tailwind CSS**
В данном проекте выбрала Tailwind CSS, чтобы изучить альтернативный, более современный подход к работе со стилями, ранее его не использовала. К тому же Tailwind CSS активно используется в индустрии, востребован на рынке и упрощает разработку.

**Redux Toolkit**
Был выбран Redux Toolkit, так как он чаще используется в коммерческих проектах и предоставляет более структурированный подход.
Также хотелось получить больше опыта и практики, используя Redux Toolkit и RTK Query.
